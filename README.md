# peanutbot

A Discord (and possibly Stoat/Revolt) bot for the AmbleLabs Discord!

## Features:
- Shortlinks: tired of telling people to screw off? Well, here you go!
  - Wiki
  - Forge
  - Eta
- Automatic Support: wow! this is practically agi!!
- Wiki Search
- meow meow meow
- Welcome People
- Dev chat argument stopper

## Setup

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run .
```
